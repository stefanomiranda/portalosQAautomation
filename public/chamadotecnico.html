4) 🛑 Checkpoint 1 - T088 no SOM

📋 Tratamento manual da T088 (Planta Externa)

  1. Acesse o SOM (Service Order Management).
  2. Localize a T088 do TT Externo ID .
  3. Selecione o tipo de encerramento:
    • Pendencia (cliente ausente) → codigo 8010 (Cliente Ausente), motivo "Cliente ausente no momento da visita".
    • Sucesso (visita resolvida) → codigo 121A1 (ONT Fonte de Alimentacao Substituida), informar matricula do tecnico, encerramento externo.
  4. Clique em Update.
  5. Se escolheu Pendencia, aguarde a Tenant reagendar (voce sera notificado).

5) 🛑 Checkpoint 2 - Patch TT V2 + T088 final

📋 Patch TT V2 (apos Tenant reagendar)

  1. A Tenant reagendou a visita - voce recebera o novo SA ID.
  2. Copie o SA ID do novo agendamento da UI do SOM.
  3. Cole o ID no campo Appointment ID abaixo.

📋 T088 final (encerre como Sucesso)

  1. Acesse o SOM novamente.
  2. Localize a T088 do mesmo TT Externo ID.
  3. Selecione Sucesso → codigo 121A1, encerramento externo.
  4. Clique em Update.
