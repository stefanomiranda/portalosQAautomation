🔌 APIs de Gerenciamento

Bloqueio Parcial

Bloqueia parcialmente um serviço de uma Ordem de Serviço existente.
🚫

Bloqueio Total

Bloqueia totalmente todos os serviços de uma Ordem de Serviço existente.
🔓

Desbloqueio

Desbloqueia um serviço ou todos os serviços de uma Ordem de Serviço existente.

Detalhes da Operação:

Atenção: A Ordem de Serviço precisa estar com a instalação concluída no FSL.

Selecione uma OS para ver os detalhes.
